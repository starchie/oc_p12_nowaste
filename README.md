# NOWASTE

# INFOS

Firebase unit test need Firebase emulator

installation :
```
curl -sL firebase.tools | bash
```

In folder of your choice : 
```
Firebase init
```

select services :
Firestore, Storage, Emulators.

adress:

Authentication  localhost:9099  
Firestore       localhost:8080  
Storage         localhost:9199
